checkNextPrice(cart, brandId, prices) → {number}

Функция определяет количество товаров определенного бренда и меняет цену текущего бренда в зависимости от activate_price_block_count. Если количество блоков больше activate_price_block_count, то берет соответствующую цену
Parameters:
Name Type Description
cart Array все товары в корзине
brandId number Id - текущего бренда
prices Array массив цен с activate_price_block_count
Returns:
новый price
Type
number
